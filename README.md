# 🚀 NextHire: The Ultimate AI-Powered Recruitment Ecosystem

**NextHire** is a cutting-edge, enterprise-grade recruitment platform designed to revolutionize the way talent meets opportunity. Developed as a high-stakes graduation project, it combines modern aesthetics with a sophisticated suite of 9+ AI-driven tools, biometric security, and a robust state-management architecture.

---

## 🌟 Vision & Design
The platform follows a **"Bright & Premium"** philosophy, moving away from legacy dark interfaces to a clean, high-contrast, and minimalist aesthetic.
- **Micro-Animations**: Smooth transitions and hover effects for a responsive feel.
- **3D Visuals**: Custom 3D assets integrated into every major feature to enhance user engagement.
- **Glassmorphism**: Subtle glass-like panels for a modern, airy layout.

---

## ✨ Comprehensive Feature Suite

### 🤵 Candidate Hub (Job Seekers)
- **🛡️ Biometric Face ID Auth**: Instant, secure login using `face-api.js` for browser-based facial recognition. No passwords required.
- **📄 AI CV Builder**: Create professional, industry-standard resumes with a guided, interactive interface.
- **🎯 AI Resume Matcher**: Real-time scoring system (ATS-style) that analyzes your resume against job descriptions to provide a match percentage.
- **🎤 Mock Interview AI**: Simulate high-pressure interviews with technical and behavioral questions generated by AI.
- **🗺️ Career Roadmap**: A step-by-step visual guide to mastering the skills needed for your target tech roles.
- **👨‍🚀 Personal Branding Assistant**: AI-driven tips and analysis to optimize your LinkedIn, portfolio, and digital presence.
- **💼 Saved Jobs & Applications**: A unified tracker to manage your career pipeline and application statuses.

### 🏢 Recruiter Hub (For Companies)
- **🏗️ Enterprise Onboarding**: A specialized registration flow with industry selection and corporate validation.
- **📊 TalentAI Pulse**: A real-time data dashboard showing market trends, popular skills, and salary benchmarks.
- **📝 Premium Job Posting**: A high-end form for companies to deploy jobs to the NextHire network with specific salary and location metrics.
- **📈 Applicant Tracking**: View and manage candidates who apply to your listings.

### 🛠️ Strategic & Utility Tools
- **🔄 Pivot Predictor**: Uses market analytics to suggest the best career transitions and "pivots" for tech professionals.
- **⚖️ Offer Weight Calculator**: A professional tool to compare multiple job offers based on salary, benefits, culture, and location.
- **⚡ Internet Speed Monitor**: A premium 3D gauge tool to test connectivity (Latency, Jitter, Stability)—critical for remote work.
- **🗺️ Service Ecosystem**: An integrated "About Us" page that connects all platform services in a single, beautiful hub.

## ⚙️ Technical Architecture

### **Frontend Stack**
- **Library**: React.js (via Vite)
- **Styling**: Pure CSS3 (Custom Design System with Variables)
- **Animation**: CSS Keyframes + Framer-style transitions
- **Biometrics**: `face-api.js` (TensorFlow.js under the hood)

### **State & Logic**
- **Context API**: `TheUserContext` manages global authentication, job data, and user preferences.
- **Form Management**: Formik + Yup for complex enterprise-grade form validation.
- **Data Persistence**: LocalStorage-driven "Mock Backend" for high-fidelity simulation without server latency.

## 🚀 Installation & Setup

1. **Clone the Repo**:
   ```bash
   git clone https://github.com/your-username/Next_Hire.git
   ```

2. **Install Packages**:
   ```bash
   npm install
   ```

3. **Run Dev Environment**:
   ```bash
   npm run dev
   ```

4. **Biometric Setup**:
   Ensure you allow camera permissions when prompted for the Face ID features to work.
